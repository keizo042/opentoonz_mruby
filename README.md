opentoonz\_mruby
===============

Usage:
------
mruby extension for opentoonz

Requirements:
-------------
cmake  
opentoonz  
c++ enviroment  

Install:
--------
when you work in PROJECT\_ROOT.  
  
mkdir build  
cd build  
cmake ../src  
make
